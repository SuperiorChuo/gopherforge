# 代码生成器

选表、配字段、一键生成 CRUD 前后端代码——新标准页面分钟级落地。入口：控制台「系统管理 → 代码生成器」。

## 三种生成模式

| 模式 | 适用 | 产物差异 |
|------|------|----------|
| **单表** | 普通实体 | 标准列表页 + 抽屉表单 + CRUD API |
| **树表** | 部门/分类等层级数据 | 树形表格、父级选择、递归接口 |
| **主子表** | 订单-明细类一对多 | 主表页内嵌子表编辑、事务化保存 |

## 使用流程

1. **选表**：从数据库反射现有表（先建表/写迁移，再生成代码）。
2. **配字段**：逐字段设置显示名、组件类型（输入/下拉/日期…）、是否列表展示/可查询/必填。
3. **预览或下载**：预览逐文件查看；下载得到 zip，按包内说明放入对应目录。

> 当前实现提供文件预览和 ZIP 下载，不会直接写入当前仓库；路由、菜单、权限和迁移仍需开发者按[二次开发](/guide/extend)完成接线。

## 产物内容

- 后端：model / store / service / api handler / 路由注册片段
- 前端：TypeScript 接口封装 + 列表页（TableToolbar + 过滤 + Table + 表单抽屉）
- 权限点与菜单：给出建议的权限码与菜单条目，按[二次开发](/guide/extend)接线

生成代码遵循仓内全部惯例（响应信封、分页参数、tenant_id、权限码格式），生成即合规。
